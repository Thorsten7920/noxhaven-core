# NOXHAVEN – FiveM RP Project

NOXHAVEN is an adult, structured FiveM RP project with a strong focus on
clean architecture, stability and long-term maintainability.

This is NOT a greenfield project. Core systems already exist and are being
refactored and extended.

## Tech Stack
- FXServer
- Lua (Client / Server)
- OX Core / ox_lib
- MySQL
- Linux (Ubuntu)

## Project Status
- Existing core structure
- Ongoing refactoring
- Modular system design
- No ESX / QB dependencies

## Looking for Collaborators
We are NOT looking for freelancers, script sellers or one-off features.

We are looking for **1–2 long-term collaborators** who want to actively
contribute to architecture, maintenance and development.

### Roles
